# Data Analyzer Skill

**数据分析技能** - 为Analysis Agent提供数据分析和可视化能力

## 快速激活

```
分析这些数据
```
```
生成数据分析报告
```
```
创建数据可视化图表
```

## 功能

1. **描述性统计** - 均值、中位数、标准差等
2. **趋势分析** - 识别数据趋势和模式
3. **对比分析** - 多组数据对比
4. **数据可视化** - 生成图表（柱状图、折线图、饼图等）

## 使用场景

- 房地产市场数据分析
- 项目投资回报分析
- 竞品数据对比
- 销售数据分析

## API

### analyze(data, analysis_type="descriptive")
执行数据分析

### visualize(data, chart_type="bar")
生成数据可视化图表

### compare(data_groups)
对比多组数据
